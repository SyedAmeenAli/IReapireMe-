Project Scope & Executive Summary

This quotation covers the full-stack design and development of the iRepairMe web application. The system will encompass a premium responsive frontend (React 19, Vite, GSAP), SMS OTP-based user authentication (Twilio Verify), cloud databases (MongoDB + Firebase Firestore), an admin control panel for price and booking management, and a tracking system for device repair tickets.

Phase / Milestone Scope & Deliverables Amount
Phase 1: Frontend & UI Design Responsive layouts (mobile & desktop), custom design system (Teal theme), smooth animations (GSAP), shopping cart, and multi-step booking stepper. ₹10,000.00
Phase 2: Authentication & SMS OTP Node.js Express backend integration, JWT-based security middleware, and Twilio SMS verification integration for passwordless logins. ₹6,000.00
Phase 3: Database Integration Database provisioning and connection (MongoDB Atlas for user schemas, Firebase Firestore for real-time order tracking and cart synchronization). ₹4,000.00
Phase 4: Admin Dashboard Admin control panel for real-time pricing updates, managing customer bookings, tracking ticket updates, and viewing analytics. ₹3,000.00
Phase 5: Deploy & Testing Multi-device QA testing, server setup (Vercel + Render + Atlas), custom domain mapping, and final handover. ₹2,000.00
